# 최후의 거점 (가칭) — 개발 기획서 v0.1

## 1. 게임 개요

| 항목 | 내용 |
| --- | --- |
| 장르 | 세로형 모바일 기지 운영 생존 전략 게임 |
| 핵심 경험 | 무한히 이어지는 필드 한가운데의 움직일 수 없는 최후의 생존자를 중심으로, 방어와 연구를 성장시켜 8방향에서 몰려오는 몬스터로부터 최대한 오래 생존한다. |
| 플레이 환경 | 모바일 웹, 세로 모드 우선 |
| 조작 | 탭으로 슬롯 선택, 건설/생산/강화 버튼 조작, 밤에는 제한된 긴급 스킬 사용 |
| 승패 | 본진 체력이 0이면 게임 종료. 생존 일수와 최고 기록을 목표로 한다. |

## 2. 핵심 원칙

1. 플레이어 캐릭터는 이동하지 않는다. 전략은 기지 배치와 경제·방어 판단에서 나온다.
2. 낮에는 준비하고 밤에는 싸운다. 밤에 준비 부족을 즉시 해결할 수 없어야 한다.
3. 단일 골드가 아닌 복수 자원이 유닛과 방어 시설의 선택을 만든다.
4. 화면은 일반적인 작은 기지 맵처럼 보이되, 모든 설치는 터치하기 쉬운 고정 슬롯에서 한다.
5. 초반은 이해하기 쉽고, 시간이 지날수록 생산 기반·강화·적 특성이 늘어난다.

## 3. 하루 진행

| 시간대 | 길이 (초기값) | 가능한 행동 | 제한 |
| --- | ---: | --- | --- |
| 낮 | 60초 | 자원 수령, 건설, 유닛 생산/회복, 수리, 연구 | 약한 위협만 존재하거나 전투 없음 |
| 황혼 | 20초 | 다음 밤 정보 확인, 마지막 배치·수리·강화 | 새 장기 생산은 제한 |
| 밤 | 60초 | 자동 전투 관찰, 긴급 스킬·제한된 긴급 수리 | 신규 건설, 일반 강화, 자유 재배치 불가 |

- 날짜가 지날수록 밤은 길어지고 적의 수·체력·속도·특수 능력이 증가한다.
- 5일 단위로 보스 또는 특수 밤(독안개, 광폭화 등)을 배치한다.

## 4. 맵과 배치

### 맵 구조

- 중앙: 건축물이 아닌 `최후의 생존자`가 고정되어 있으며, 이곳이 패배 조건과 자동 골드 수입의 원천이다.
- 주변: 북, 북동, 동, 남동, 남, 남서, 서, 북서에서 몬스터가 접근하는 무한 필드.
- 별도 공격로·건축 전용 슬롯은 없다. 유닛, 포탑, 연구 건물은 필드의 빈 위치 어디든 설치한다.
- 선택한 설치물은 60px 단위 격자에 맞춰 배치하며, 다른 설치물과 너무 가까운 곳에는 놓을 수 없다.

### 모바일 UI

- 상단: 현재 일수, 시간대, 본진 체력, 자원
- 중앙: 방사형 기지 맵과 전투 상황
- 하단: 평상시에는 작게 접힌 건설 바를 두고, 탭하면 설치 패널을 연다.
- 설치 패널 탭: `유닛`, `포탑`, `업그레이드`, `건축물`
- 공격로별 위험도는 색상 또는 경고 아이콘으로 표시한다.

### 설치 조작 규칙

모든 설치 항목은 **탭 설치**와 **드래그 설치**를 함께 지원한다.

| 조작 | 동작 | 목적 |
| --- | --- | --- |
| 항목 탭 | 설치 가능한 기본 추천 슬롯을 강조하고, 해당 슬롯을 다시 탭하면 즉시 설치 | 빠른 조작 |
| 항목 길게 누르기/드래그 | 손가락 아래에 반투명 미리보기 모델을 표시한다. 설치 가능 슬롯 위에서 놓으면 설치한다. | 정확한 위치 선택 |
| 드래그 중 | 공격 범위, 공격 방향, 효과 범위(광역/덫)를 가이드 라인으로 표시한다. | 배치 결과 예측 |
| 설치 불가 슬롯 | 빨간색으로 표시하고 자원 부족·건물 조건 부족 등의 이유를 짧게 표시한다. | 실수 방지 |

- 유닛·포탑: 드래그 중 원형 또는 공격로 방향의 반투명 사거리 표시.
- 관통/직선형 포탑: 사거리 원 대신 진행 방향 가이드 라인 표시.
- 범위 포탑·폭발 덫: 피해 반경을 별도 원으로 표시.
- 근접 유닛: 사거리보다 적을 저지하는 위치와 막을 수 있는 적 수를 강조.
- 건축물: 본진 주변의 건축 전용 슬롯만 강조.
- 업그레이드 탭: 해금된 강화와 필요한 연구 건물을 보여주는 **현황/안내 탭**이다. 실제 구매는 연구 건물을 탭해 열린 모달에서만 가능하다.

## 5. 경제 건물과 자원

초기 버전에서 골드는 최후의 생존자에게 자동으로 들어온다. 플레이어는 광산·농장 같은 경제 기지를 따로 건설하지 않으며, 받은 골드를 방어와 연구에 집중 투자한다.

| 건물 | 기본 생산/기능 | 주요 사용처 | 역할 |
| --- | --- | --- | --- |
| 최후의 생존자 | 매초 골드 자동 수입 | 유닛, 포탑, 덫, 연구 건물 | 기본 경제 |
| 훈련소 | 유닛 연구 해금 | 유닛 공격/체력/사거리 강화 | 성장 기반 |
| 야전 공방 | 포탑·덫 연구 해금 | 포탑/덫 강화 | 방어 성장 |
| 의무소 | 유닛 회복 지원 | 회복 속도, 부상 유닛 지원 | 유지력 |
| 보급소 | 유닛 고용 지원 | 고용 비용 절감, 보급 강화 | 운영 효율 |

- 각 건축물 종류는 필드에 최대 3개까지만 설치할 수 있다.

- 몬스터 처치와 보상에서도 소량의 골드 또는 철을 획득한다.
- 식량은 첫 MVP에서 별도의 상시 자원으로 표시하지 않는다. 휴식소가 유닛의 휴식·보급을 추상화한다.
- 이후 식량 관리의 재미가 필요해질 경우에만 농장/식량 자원을 추가한다.
- 경제 건물은 밤에 공격받아 파괴될 수 있다. 일부 적은 경제 건물을 우선 공격한다.

### 경제 건물 선택 및 업그레이드

낮과 황혼에 맵 위의 경제 건물을 탭하면 **건물 정보 패널**이 열린다. 패널에는 현재 레벨, 다음 생산 주기, 내구도, 업그레이드 비용과 다음 레벨 효과를 표시한다.

| 건물 | 레벨 1 | 레벨 2 | 레벨 3 |
| --- | --- | --- | --- |
| 광산 | 8초마다 골드 5 | 8초마다 골드 8 | 골드 10 + 철 1 |
| 벌목소 | 10초마다 나무 4 | 10초마다 나무 7 | 나무 10 + 모든 건축 나무 비용 -10% |
| 휴식소 | 낮마다 유닛 체력 20% 회복 | 낮마다 40% 회복 | 완전 회복 + 유닛 유지비 -25% |

- 업그레이드는 즉시 완료하되, 낮/황혼에만 가능하다.
- 건물이 파괴되면 해당 생산·회복 효과가 멈추며, 수리하거나 다시 지어야 한다.

## 6. 건축물

| 건물 | 역할 | 주요 강화 예시 |
| --- | --- | --- |
| 본진 | 패배 조건, 핵심 스킬 제공 | 최대 체력, 보호막, 긴급 수리 횟수 |
| 광산 | 골드·철 자동 생산 | 생산량, 철 생산 해금 |
| 벌목소 | 나무 자동 생산 | 생산량, 건설 비용 할인 |
| 휴식소 | 유닛 회복·유지 지원 | 회복량, 유지비 절감 |
| 병영 | 근접 유닛 생산 | 창병 해금, 근접 유닛 체력 보너스 |
| 사격장 | 궁수 생산 | 고급 궁수 해금, 궁수 사거리 보너스 |
| 훈련소 | 유닛 전투 연구 | 공격력, 체력, 공격 간격, 사거리 강화 |
| 연구소 | 포탑·덫 연구 | 포탑 공격력, 사거리, 덫 재장전 강화 |
| 마법 연구소 | 2차 버전에서 마법 유닛/원소 포탑 생산 | 상태 이상, 마법 강화 |
| 공방 | 포탑·덫·방벽 생산 | 수리 비용 감소, 고급 방어 시설 해금 |

### 건축 구역, 파괴, 재건

- 건축물은 필드 아무 곳이 아니라, 최후의 생존자 주변에 미리 정해진 **건축 구역**에서만 운영한다.
- 게임 시작 시 기본 건축물은 각 구역에 설치되어 있다. 파괴되면 해당 구역은 빈 재건 슬롯이 된다.
- 빈 재건 슬롯을 탭하고 건축물 카드를 선택하면, 최후의 생존자가 생산한 골드를 지불해 그 자리에 다시 설치한다.
- 같은 종류의 건축물은 최대 3개까지만 유지할 수 있다. 파괴된 건축물은 이 제한에서 즉시 제외된다.

### 건축물 체력, 수리, 업그레이드

건축물을 탭하면 해당 건축물 전용 정보 모달을 연다. 모달에는 현재 체력, 레벨, 고유 선택지와 수리/업그레이드 버튼을 표시한다.

| 행동 | 비용 | 효과 |
| --- | --- | --- |
| 수리 | 철근 | 잃은 체력 회복 |
| 업그레이드 | 골드 | 최대 체력 및 고유 효과 강화 |
| 재건 | 골드 | 파괴된 빈 구역에 건축물 다시 설치 |

- 수리는 낮·황혼에만 가능하며, 밤에는 긴급 수리 같은 제한된 예외만 허용한다.
- 업그레이드는 훈련소의 연구 단계, 의무소의 회복량, 보급소의 고용 할인처럼 건축물별 효과를 높인다.
- 철근은 최후의 생존자가 생산하는 수리 전용 자원이다. 필드나 몬스터 보상에서는 얻지 않으며, 중앙 기지를 탭해 비축된 철근을 회수한다.
- 건축물의 업그레이드 레벨에는 최대치가 없다. 대신 다음 레벨의 골드 비용은 점점 증가한다.
- 일반 레벨 보너스는 작고 누적되며, 5레벨 단위의 이정표에서는 새로운 효과나 큰 보너스를 제공한다.

| 건축물 | 모달에서 선택하는 고유 기능 예시 |
| --- | --- |
| 훈련소 | 검사 체력, 궁수 사거리, 유닛 공격력 연구 |
| 야전 공방 | 포탑 공격력, 포탑 사거리, 덫 재장전 연구 |
| 의무소 | 유닛 회복, 회복 범위, 부상 회복 속도 강화 |
| 보급소 | 유닛 고용 할인, 포탑 건설 할인, 긴급 보급 선택 |

#### 무한 레벨업 예시: 훈련소

| 훈련소 레벨 | 기본 효과 | 이정표 효과 |
| --- | --- | --- |
| Lv.1 | 기본 유닛 생산 가능 | — |
| Lv.2 | 모든 유닛 공격력 +5% | — |
| Lv.3 | 모든 유닛 공격력 +10% | — |
| Lv.5 | 모든 유닛 공격력 +20% | 유닛 공격 속도 +5% |
| Lv.10 | 모든 유닛 공격력 +45% | 유닛 탐지 범위 +1 |

- Lv.11 이후에도 공격력 보너스와 비용은 계속 상승한다.
- 예시 비용 공식: `다음 레벨 골드 = 기본 비용 × 1.18^(현재 레벨 - 1)`.

## 8. 강화와 연구

강화는 기지에 해당 연구 건물을 설치한 뒤에만 이용할 수 있다. 예를 들어 공격력 강화는 훈련소가 있어야 구매할 수 있다.

### 연구 건물 모달

낮과 황혼에 훈련소·연구소 같은 연구 건물을 탭하면 해당 건물의 **연구 모달**을 연다.

- 상단: 건물 이름, 레벨, 내구도, 다음 레벨업 비용
- 중단: 해당 건물에서 진행 가능한 연구 목록
- 각 연구 카드: 효과, 비용, 선행 조건, 현재 단계, 구매 버튼
- 잠긴 연구: 필요한 건물 레벨 또는 선행 연구를 명시
- 밤에는 모달을 열어 정보는 볼 수 있지만, 연구 구매는 할 수 없다.

강화는 두 층으로 나눈다.

1. **건물 레벨업:** 생산 건물의 해금과 생산량을 높인다. 예: 병영 2레벨에서 창병 해금.
2. **연구 강화:** 설치된 연구 건물에서 구매하는 전체 전략 효과다. 예: 궁수 사거리 +1, 포탑 공격 간격 -15%.

강화 예시:

- 모든 근접 유닛 최대 체력 +20%
- 궁수 공격이 적 1명을 관통
- 화살 포탑 공격 간격 -10%
- 다음 포탑 건설 골드 -30% (1회)
- 목책 파괴 시 주변 적에게 피해
- 휴식소 유닛 회복량 +15%

| 연구 건물 | 가능 강화 예시 |
| --- | --- |
| 훈련소 | 모든 유닛 공격력 +10%, 근접 유닛 체력 +20%, 궁수 사거리 +1 |
| 연구소 | 모든 포탑 공격 간격 -10%, 덫 재장전 -15%, 포탑 내구도 +25% |
| 공방 | 방벽 최대 체력 +20%, 수리 비용 -15%, 폭발 덫 피해 +20% |

낮과 황혼에만 연구 구매가 가능하며, 밤에는 본진 긴급 스킬만 사용 가능하다.

## 9. 필수 데이터 모델

데이터는 **종류의 기본 정의**와 **게임 중 개체 상태**를 분리한다.

```ts
type ResourceCost = {
  gold?: number;
  wood?: number;
  iron?: number;
};

type CombatStats = {
  maxHp: number;
  attackDamage: number;
  attackInterval: number; // 다음 공격까지의 시간(초)
  attackRange: number;
  moveSpeed: number;
  armor: number;
};

type UnitDefinition = CombatStats & {
  id: string;
  tags: string[];
  cost: ResourceCost;
  restCapacityCost: number;
  blockCount: number;
  requiredBuilding: string;
  requiredBuildingLevel: number;
};

type BuildingDefinition = {
  id: string;
  maxLevel: number;
  buildCost: ResourceCost;
  unlocks: string[];
  effectsByLevel: Record<number, Effect[]>;
};

type UpgradeDefinition = {
  id: string;
  cost: ResourceCost;
  requirements?: string[];
  effects: Effect[];
};
```

- `Definition`은 검사, 궁수, 화살 포탑 같은 종류의 기본값이다.
- 실제 배치된 개체는 `currentHp`, `position`, `cooldown`, `level`, `buffs` 같은 런타임 상태를 별도로 가진다.
- 건물과 연구의 `Effect`를 모아 최종 능력치를 계산한다. 기본 데이터를 직접 수정하지 않는다.

## 10. MVP 개발 범위

첫 플레이 가능한 버전에는 아래만 구현한다.

1. 세로 화면의 중앙 본진과 8개 공격로
2. 낮/황혼/밤 타이머와 날짜 상승
3. 골드·나무·철 자원
4. 광산·벌목소·휴식소·병영·공방·훈련소 및 기본 자동 생산/회복
5. 검사·궁수, 화살 포탑·가시 덫·목책
6. 일반·질주·거대 몬스터
7. 본진 체력, 자동 전투, 패배 및 생존 일수 표시
8. 훈련소 모달에서 구매하는 8~12개의 기본 유닛 강화
9. 하단 4탭 설치 패널과 탭 설치·드래그 설치·사거리 미리보기

## 11. MVP 이후 확장

- 마법 연구소, 마법사, 원소 포탑, 마력 자원
- 보스와 특수 밤
- 영웅/본진 스킬 트리
- 유닛 특성·장비·등급
- 도감, 업적, 영구 해금 요소
- 일일 도전과 점수 경쟁

## 12. 현재 결정이 필요한 항목

1. 세계관: 판타지, 좀비 아포칼립스, 식물/정령 방어 중 무엇을 택할지
2. 한 판의 목표: 무한 생존 기록형인지, 특정 일수 생존 후 탈출하는 클리어형인지
3. 유닛 조작: 완전 자동 전투인지, 낮에만 공격로 재배치가 가능한지
